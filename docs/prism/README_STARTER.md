# Prism

> **One concept. As many ways as it takes to make it click.**

Prism is an adaptive Chrome learning copilot that understands learner-selected material, asks what the learner wants to accomplish, and teaches the concept through interactive methods that can change when the learner gets stuck.

## Start here

- Read [`PRISM_HACKATHON_BUILD_SPEC.md`](./PRISM_HACKATHON_BUILD_SPEC.md)
- Coding agents must read [`AGENTS.md`](./AGENTS.md)

## Hackathon vertical slices

- **Prism School:** Guided linear-equation homework help with attempt-before-answer gating and a Visual Lab mode.
- **Prism Life:** Manual compound-interest and introductory ETF learning with interactive simulations.
